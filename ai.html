Steps (Size: 2.6MB)
  1. This transparent character video is created from previous video. We can see significant reduction in size(From 18.6MB to 2.6MB).
  2. Open new or existing Project in Adobe Character Animator
  3. Create or Import Puppet in project and create scene.
  4. Import Audio File(mp3) and Drag into scene.
  5. Select background color 'GREEN'.
  6. Under Record Section, Click on timeline > compute lip sync from audio. it will make lip sync on character based on audio.
  7. We can customize puppet's eye, face and body movement.
  8. Export to desire location. Output will be in MP4 format.
  9. Download FFmpeg and unzip it. There will be ffmpeg.exe file under bin folder. We can set this location in environment variable to be able to execute ffmpeg command globally.
    Note: if you don't want to do in local machine, we can use Google Colab
  10. Run Command: ffmpeg -i input_video.mp4 -vf "chromakey=green:similarity=0.087:blend=0.0" -c:v libvpx-vp9 -pix_fmt yuva420p output_video.webm
  11. By Adjusting blend and similarities values, we can fine-tune the chroma key effect based on the specific characteristics of our video
  12. Make sure to replace correct input and output file locations. Output video extension will be in webm
